set_option

set option in option list

Syntax 

 value = set_option(option_list,option,value)
 value = set_option(option_list,{option1,option2,option3})

Input 

option_list   - Cell Array
option        - String
value         - some type

Output 

out_list      - Cell Array

See also

check_option, get_option, delete_option
