LAMP-Solar - MATLAB toolbox for solar cell analysis

Overview
--------

LAMP-Solar is a MATLAB toolbox containing the following features


For an introduction, please have a look at the help first. If the
toolbox is installed the help can be found in the MATLAB help window
which can be opened by typing 'doc LAMP-Solar'.
For installation instructions, you can also refer to the file INSTALL
in this directory.

Directory structure
-------------------
AUTHORS		     information about the authors of LAMP-Solar
ChangeLog	     a short version history
COPYING		     information about redistributing LAMP-Solar
data (dir)           some sample data sets
examples (dir)       simple examples for using LAMP-Solar
geometry (dir)       geometry tools
help (dir)	     user and developer documentation
info.xml             used by MATLAB
INSTALL              installation instructions
MAKEFILE             contains the compiling options !! to be edited by the advanced user !!
LAMPsolar_settings   contains the configuration of LAMP-Solar !! to be edited by the user !!
qta (dir)            core scripts for working with ODFs, EBSD data, and PDFs
README               this file
startup.m            installation routine automatically called by MATLAB if the
                     toolbox is contained in the search path of MATLAB
startup_LAMPsolar    installation routine for LAMP-Solar - called manually or by startup.m
startup_root         used for the root installation
tests (dir)          a self test
tools                auxiliary tools


Feedback
--------
Your comments are welcome! LAMP-Solar is heavily developed and may not be
as robust or well documented as it should be. Please keep track of bugs or
missing/confusing instructions and report them to

  Group homepage: http://lamp.pitt.edu

  LAMP-Solar Homepage: http://code.google.com/p/lamp-solar/

If you find LAMP-Solar useful, we would be delighted to hear about what application you are using LAMP-Solar for!
